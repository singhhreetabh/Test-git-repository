## Python Calculator with Advanced GUI

This repository contains a Python calculator application with an advanced graphical user interface built using Tkinter.

### Features
- Basic arithmetic operations (+, -, *, /)
- Scientific functions (sin, cos, tan, log, sqrt)
- Constants (pi, e)
- Exponentiation (^)
- Calculation history
- Clean, modern UI

### Usage
1. Ensure Python is installed.
2. Run the calculator:
   ```
   python calculator.py
   ```
   Or with the virtual environment:
   ```
   c:/Users/Hreetabh/Documents/Repositories/Test-git-repository/.venv/Scripts/python.exe calculator.py
   ```

### Requirements
- Python 3.x
- Tkinter (usually included with Python)
